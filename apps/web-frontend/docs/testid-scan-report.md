## E2E з”ЁдҫӢдёҺ testid еҜ№еә”пјҲз”өжәҗз®ЎзҗҶпјү

- **е”ҜдёҖжҖ§**пјҡйқҷжҖҒ testid еңЁеҚ•йЎөеҶ…жҜҸдёӘ id д»…еҮәзҺ°дёҖж¬ЎпјҲжҲ–жҢүи®ҫи®ЎеӨҡе®һдҫӢз”Ё `.first()` еҸ–йҰ–пјүгҖӮеҠЁжҖҒ testidпјҲеҰӮ `e2e-device-${driverType}-conn`гҖҒ`ui-app-menu-device-${driverType}`пјүдҫқиө–иҝҗиЎҢж—¶ж•°жҚ®пјҢеҗҢдёҖж—¶еҲ»дёҚеҗҢе®һдҫӢ id дёҚеҗҢпјҢи§ҶдёәвҖңйҖ»иҫ‘е”ҜдёҖвҖқгҖӮ
- **з”өжәҗз®ЎзҗҶ**пјҡ`ui-power-manager-root`гҖҒ`ui-app-power-page-output-power-1/2`гҖҒ`ui-app-power-page-restart`гҖҒ`ui-app-power-page-shutdown`гҖҒ`ui-app-power-page-force-update`гҖҒ`ui-app-menu-open-power-manager` д»ҘеҸҠзЎ®и®Өеј№зӘ— `ui-confirm-dialog-root`гҖҒ`ui-confirm-dialog-btn-cancel`гҖҒ`ui-confirm-dialog-btn-confirm` е’Ң `tb-act-toggle-navigation-drawer`гҖҒ`ui-app-menu-drawer` еңЁ Power еҠҹиғҪиҢғеӣҙеҶ…е…ЁеұҖе”ҜдёҖпјҢеҸҜзӣҙжҺҘдҪңдёә E2E е®ҡдҪҚж ҮеҮҶгҖӮ
- й’ҲеҜ№е…·дҪ“ spec зҡ„еј•з”Ёжё…еҚ•дёҺи§ҰеҸ‘ж–№ејҸж ЎйӘҢи§Ғ **testid-validation-report.md** дёӯгҖҢжҢү E2E з”ЁдҫӢзҡ„ testid ж ЎйӘҢгҖҚгҖӮ
